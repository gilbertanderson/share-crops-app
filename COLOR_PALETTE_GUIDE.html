Usage Guidelines

✓ Primary Color (#2da965)

  • Call-to-action buttons
  • Active navigation states
  • Focus rings and highlights
  • Success confirmations
  • Brand identity elements

✓ Light Green (50-200)

  • Hover states for buttons
  • Background colors
  • Alert/notification backgrounds
  • Secondary interactions

✓ Dark Green (600-900)

  • Hover and active states
  • Dark mode variants
  • High-contrast text
  • Disabled state indicators

✓ Neutral Gray (0-900)

  • Page backgrounds (0, 50)
  • Body text (700, 800, 900)
  • Borders and dividers (200, 300)
  • Secondary text (400, 500, 600)
  • Disabled states

Design System Version 1.0

Color palette created with accessibility and modern design standards in mind. All colors meet WCAG AA contrast ratios for readability.
